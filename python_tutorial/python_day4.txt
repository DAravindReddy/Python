1. What is string in python?
Ans: A string is a sequence of characters.
Eg: name = "Aravind"
or 
name = 'Aravind'

2. String indexing?
Ans: Every character has a position(index).
Eg: 
name = 'Aravind'

print(name[0])	 #A
print(name[-1])	 #d last character

3. String slicing
Ans: used to get part of string
syntax: string[start:end]

Eg:
name = "Aravind"

print(name[0:4]) #Arav
print(name[4:7]) #ind
print(name[:3])  #Ara
print(name[3:])  #vind
